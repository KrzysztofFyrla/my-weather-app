# My Weather App
![Login panel](./src/main/resources/public/css/MyWeatcherApp.png)

My Weather App is a simple forecast app, which uses some APIs to fetch forecast data from the OpenWeather.

## Technologies
Project is created with:
* JAVA version 8
* Maven
* Vaadin
* OpenWeather API
